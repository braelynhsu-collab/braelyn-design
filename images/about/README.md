放置 About 页面作者照片：

- 推荐文件名：`braelyn-portrait.jpg`
- 当前页面先使用占位图：`images/about/portrait-placeholder.svg`
- 你放入真人照片后，把 `about.html` 中的 `images/about/portrait-placeholder.svg` 改成 `images/about/braelyn-portrait.jpg` 即可；也可以直接把照片发我，我来替你接上。
- 推荐比例：竖图，约 4:5 或 3:4；半身、侧身或有环境感的职业照片都可以。
- 展示形式：页面会自动裁切成 editorial 拱形竖幅，不会显示成普通方形头像。

如果你使用 PNG/WebP，需要同步修改 `about.html` 里的图片文件名。
