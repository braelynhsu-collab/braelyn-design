# Objekt 物与空间项目

## 文件夹说明
请将此项目的图片放入此文件夹。

## 建议的图片命名规则
- `cover.jpg` - 项目封面图（必须）
- `detail-01.jpg` - 细节图1
- `detail-02.jpg` - 细节图2
- `process-01.jpg` - 过程图

## 图片尺寸建议
- 封面图：宽度至少2400px，推荐比例 16:9
- 细节图：宽度至少1600px
- 所有图片格式：JPG或PNG

## 如何更新
1. 将图片放入此文件夹
2. 更新 `projects.json` 中的 `coverImage` 路径
3. 刷新网站查看效果
